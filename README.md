# Project2_DSCI311
Project 2 for DSCI 311: Principles &amp; Techniques of Data Science @ UO
Investigates features of speech proxying for vowel characteristics (aggregated spectral centroid and bandwidth for a given audio file) in Japanese, English, and Mandarin Chinese.
Data from [SPREDS-D2](https://ast-astrec.nict.go.jp/en/release/SPREDS-D2/), an open-source repo of speech recordings (wav files) in the 3 languages.
